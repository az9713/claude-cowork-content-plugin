# Content Repurposing Plugin - All Commands

Here are all available commands and skills in this plugin:

## Commands (Quick Actions)
- `/content-repurposing:extract` - Extract content ideas from any source into a structured table
- `/content-repurposing:linkedin` - Generate LinkedIn posts from your content
- `/content-repurposing:twitter` - Generate Twitter/X threads from your content
- `/content-repurposing:quotes` - Extract quotable statements from your content
- `/content-repurposing:titles` - Generate compelling titles for your content

## Skills (Auto-invoked by Claude)
- **content-extraction** - Full content idea extraction across all platforms
- **key-takeaways** - Summarize key takeaways into bullet lists
- **linkedin-post** - Create LinkedIn posts
- **twitter-thread** - Create Twitter/X threads
- **quote-extractor** - Pull quotable statements
- **substack-note** - Generate Substack Notes
- **title-generator** - Generate titles across platforms

Use any command with a file reference or paste content directly.
